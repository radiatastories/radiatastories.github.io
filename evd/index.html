Radiata Stories EVD Commands

Generated by Claude Opus 4.6
Derived from CRadiScript::Command_XX MIPS handlers in the Radiata Stories DEBUG PS2 ELF.
Base word layout: [execute:8][arg2:8][p_len:8][opcode:8] — little-endian.   Bit 7 of execute = continue same tick (no yield).   Returns: 0 = continue • 1 = yield • 2 = jump taken • <0 = error

Example: 10opcode 01payload length 00arg2 80execute 40 1F 00 00payload
Opcode—0x10 (SET_FLAG), payload length—1, arg2—0, execute—0x80, payload—flag 8000 set to disable.
